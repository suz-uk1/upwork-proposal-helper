Payments / Pro status

If you purchased Pro but the extension still shows Free:

  1. Open the extension → Settings
  2. Click Re-sync Pro
  3. If it still fails, click Enter license key again and retry

If it still doesn’t work, contact support with the email you used at checkout.

Extraction issues

  • If extraction is partial, paste the job description manually into the textarea.
  • Upwork pages change frequently. If it breaks, re-open the job detail page and click Re-extract.
